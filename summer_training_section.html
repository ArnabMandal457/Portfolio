Capstone project: PookiePrank — a Python + Netcat-based reverse shell simulator built to demonstrate real social engineering and backdoor attack vectors in a controlled, educational setting. The tool silently deploys a netcat listener via a batch payload, establishes a reverse shell over a local network, and hands the attacker a live command session — wrapped in a disarming UI to simulate how trust exploitation actually works. Covers Nmap-based target discovery (CIDR scanning), reverse shell mechanics, payload delivery, and post-connection command execution. Built for cybersecurity awareness training — making attack vectors impossible to ignore by making them personal.
